Description
    Computes the chemical potential of a system with
    eigenvalues W and temperature BETA in order for the
    system to have exactly NUMPART particles
Parameters
[in]N
    Number of eigenstates
[in]W
    Eigenstates of the system
[in]BETA
    Inverse temperature
[in,out]NUMPART
    Number of particle of the system
    On entry the desired one
    On exit the number of particles
    with the computed chemical potential
[in]STAT
      Statistics of the particles
    1 = Bosons, -1 = Fermions
[out]MU
      Computed chemical potential

0.1
