Description
    Returns the Hamiltonian of a periodic two-dimensional system
    with homogeneous on-site energies on a lattice
    The nearest-neighbour couplings are distributed accorging a 
    specific geometry GMT.
param[in] NPX 
    INTEGER
    Number of sites along the x-direction
param[in] NPY 
    INTEGER
    Number of sites along the y-direction
param[in] BCX 
    INTEGER
    OPTIONAL, Boundary condition along the x-direction 
    Default = Open boundary conditions
    BCX= 1 -> Periodic
param[in] BCY 
    INTEGER
      OPTIONAL, Boundary condition along the y-direction 
    Default = Open boundary conditions
    BCY= 1 -> Periodic
param[in] ONSITE 
    NOT IMPLEMENTED YET
param[in] GMT 
    CHARACTER
    Specifies the geometry of the system
    sq = Square, tr = Triangular, grp = Graphene
param[out] HAM 
    DOUBLE PRECISION, rank-2 array
    On exit the Hamiltonian of the system
    The 2D system has been mapped to 1D one
    according to the mapping
    (ix,iy) ->  ix+NPX*(iy-1)
