Description
    Computes the collisional integral of the GKBA master equation
    at time TIME given the GKBA evolution operator Y and
    the total self-energy components SLFL and SLFG
param[in] DTIME 
    DOUBLE PRECISION
    The time step
param[in] RO 
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index are the spatial/momentum/spin ones
    The time-dependent density matrix of the system
param[in] TIME 
    INTEGER
    The current time
param[in] Y 
      DOUBLE COMPLEX, rank-3 array
      The first index if the time one
      The second and thid index are the spatial/momentum/spin ones
      The evolution operator in the GKBA approach
param[in] SLFL 
      DOUBLE COMPLEX, rank-3 array
      The first index if the time one
      The second and thid index are the spatial/momentum/spin ones
      The LESSER component of the many-body self-energy
      computed in the Second Bord approximation
Parameters
[in]SLFG
      DOUBLE COMPLEX, rank-3 array
      The first index if the time one
      The second and thid index are the spatial/momentum/spin ones
      The GREATER component of the many-body self-energy
      computed in the Second Bord approximation
param[out] CINT 
    DOUBLE PRECISION, rank-2 array
    On exit the collisional integral at time TIME
