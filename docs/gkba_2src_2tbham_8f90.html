Description
    Returns the tight-binding free Hamiltonian
    for a 1D system with values of the couplings
    given by NNHOP
Parameters
[in]NPX
    The spatial dimension of the system
[in]BC
    OPTIONAL, Boundary conditions 1 = Periodic
[in]ONSITE
    OPTIONAL, Onsite energies
    If not provided they are set to zero
[in]NNHOP
    1D array with the nearest-neighbour coupling
    NNHOP(i) is the coupling bewteen sites i and i+1
    The element NPX is the coupling between
    sites 1 and NPX in case BC = 1
[out]HAM
    On exit the tight-binding Hamiltonian of the system

0.1
