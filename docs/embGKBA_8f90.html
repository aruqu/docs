Description
    Computes the LESSER and GREATER components of the embedding 
    self-energies. It assumes the lead is at equilibrium and its
    state is describe by a Gibbs state with
    a given inverse temperature BETA and chemical potential MU
Parameters
[in]DTIME
    DOUBLE PRECISION
    The time step
[in]NTIME
    INTEGER
    Number of time steps
[in]NS
    INTEGER
    Size of the spatial/momentum/spin configuration space
    of the system
[in]BETA
    DOUBLE PRECISION
    Inverse temperature of the lead
[in]MU
    DOUBLE PRECISION
    Chemical potential of the lead
[in]GC
    DOUBLE PRECISION
    System-lead coupling strenght 
    it is the entry of the coupling Hamiltonian
[in]GAMMA
    DOUBLE PRECISION, rank-2 array
    The adjaciency matrix of the system-lead coupling
[out]EMBSLFL
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index coincide with those of GAMMA
    On exit the LESSER component of the embedding self-energy
[out]EMBSLFL
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index coincide with those of GAMMA
    On exit the GREATER component of the embedding self-energy
