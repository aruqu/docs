Description
    Computed the LESSER and GREATER components of the
    self-energy in the Second Born approximation and within
    the GKBA limit
Parameters
[in]DTIME
    DOUBLE PRECISION
    The time step
[in]RO
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index are the spatial/momentum/spin ones
    The time-dependent density matrix of the system
[in]TIME
    INTEGER
    The current time
[in]Y
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index are the spatial/momentum/spin ones
    The evolution operator in the GKBA approach
[in]V
      DOUBLE PRECISION, rank-2 array
      A matrix containing the two-body density-density interaction
[in]VDRIVE
      DOUBLE PRECISION, rank-1 array
      OPTIONAL, An array with a temporal profile
      To be used to avoid sudden switching on of the interaction
[out]SLFL
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index are the spatial/momentum/spin ones
    On exit the LESSER component of the many-body self-energy
    computed in the Second Bord approximation
[out]SLFG
    DOUBLE COMPLEX, rank-3 array
    The first index if the time one
    The second and thid index are the spatial/momentum/spin ones
    On exit the GREATER component of the many-body self-energy
    computed in the Second Bord approximation
